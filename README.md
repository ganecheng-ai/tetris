# 俄罗斯方块 - Tetris

经典的俄罗斯方块游戏，使用 Python 和 Pygame 开发。

## 游戏特性

- 精美的游戏界面和方块设计
- 10 个关卡，难度递增
- 简体中文界面
- 经典操作方式
- 影子方块提示
- 下一个方块预览
- 暂存方块功能（Hold）
- 连击（Combo）系统
- 多种游戏主题（经典、暗黑、明亮、霓虹）
- 方块皮肤系统（经典、霓虹、粉色、黄金、冰雪、火焰、森林、彩虹、水晶）
- **双人对战模式** - 本地双人对战，互相攻击
- **多种游戏模式** - 经典、无尽、竞速、限时、大师、禅、挑战、自定义、每日挑战
- **成就系统** - 30 个成就等待解锁
- **数据统计** - 详细的游戏统计信息
- **在线排行榜** - 上传分数到全球排行榜
- **云端存档** - 保存和加载游戏进度
- **玩家系统** - 唯一玩家 ID 识别
- **自定义规则** - 自定义游戏规则，打造专属玩法
- **每日挑战** - 每天特殊的挑战任务（v2.7.0 新增）
- **回放系统** - 录制和回放精彩操作（v2.7.0 新增）
- **快捷键自定义** - 允许玩家自定义按键（v2.7.0 新增）
- **游戏速度调节** - 允许玩家调整游戏速度倍数 0.5x-2.0x（v2.7.0 新增）
- **游戏内提示** - 新手引导提示（v2.7.0 新增）
- **音效包系统** - 支持 4 种音效包（经典 8 位、电子、自然、钢琴）（v2.8.0 新增）
- **更多游戏主题** - 新增 4 种主题（海洋、星空、赛博朋克、复古）（v2.8.0 新增）
- **更多方块皮肤** - 新增 4 种皮肤（钻石、岩浆、星空、金属）（v2.8.0 新增）

## 游戏模式

### 经典模式
经典的俄罗斯方块玩法，10 个关卡挑战。每消除一定行数后升级，速度越来越快。

### 无尽模式
无限挑战模式，没有终点。每消除 10 行升一级，速度会逐渐加快，考验你的持久力！

### 竞速模式
以最快速度消除 40 行。时间就是你的对手，看看你能在多快的时间内完成挑战！

### 限时模式
在 2 分钟的时间内获得尽可能高的分数。时间结束后游戏结束，考验你的得分效率！

### 大师模式（v2.5.0 新增）
20 层关卡极限挑战，速度更快，难度更高。专为方块高手准备！

| 关卡 | 名称 | 速度 (ms) | 消除行数 |
|------|------|-----------|----------|
| 1-10 | 新手到大师 | 800-100 | 5-50 |
| 11-20 | 宗师到神王 | 80-20 | 55-100 |

### 禅模式（v2.5.0 新增）
无压力放松体验，固定慢速，没有游戏结束。适合休闲娱乐或练习技巧。

### 挑战模式（v2.5.0 新增）
特殊规则挑战，每次随机选择一种挑战类型：
- **无暂存** - 禁止使用暂存方块
- **无影子** - 不显示影子方块
- **盲目** - 不显示下一个方块
- **反向重力** - 方块从下往上出现
- **快速下落** - 方块自动快速下落
- **随机旋转** - 旋转方向随机

### 自定义模式（v2.5.0 新增）
自定义游戏规则，打造专属玩法。可自定义：
- 初始关卡和最大关卡
- 重力类型（普通/快速/随机）
- 目标消除行数
- 时间限制
- 是否启用暂存/影子/下一个方块
- 旋转系统
- 是否使用 7-bag 随机系统

### 每日挑战（v2.7.0 新增）
每天特殊的挑战任务，每天自动更新。挑战类型包括：
- **速度挑战** - 在更快速度下完成游戏
- **盲目挑战** - 不显示下一个方块
- **无暂存挑战** - 禁止使用暂存方块
- **高分挑战** - 获得尽可能高的分数
- **消行挑战** - 消除指定行数
- **连击挑战** - 达成高连击数

完成每日挑战可解锁特殊成就！

### 双人对战模式
本地双人对战，通过消除行数向对手发送垃圾行。先让对方无法放置方块者获胜！

**攻击规则：**
- 消除 2 行：发送 1 行垃圾
- 消除 3 行：发送 2 行垃圾
- 消除 4 行（Tetris）：发送 4 行垃圾
- 连击奖励：额外发送最多 5 行垃圾

## 操作说明

### 单人模式
- **← →** - 左右移动
- **↑** - 旋转方块
- **↓** - 软降（加速下落）
- **空格** - 硬降（直接落底）
- **C** - 暂存方块（Hold）
- **P** - 暂停/继续
- **R** - 重新开始
- **T** - 切换主题
- **+/-** - 游戏速度调节（v2.7.0 新增）
- **1** - 降低音效音量
- **2** - 增加音效音量
- **3** - 降低音乐音量
- **4** - 增加音乐音量
- **M** - 切换背景音乐播放/暂停
- **F5** - 刷新在线状态
- **S** - 保存游戏（暂停时）
- **L** - 加载存档
- **DEL** - 删除存档（暂停时）

### 双人对战模式

**玩家 1（左侧）：**
- **W** - 旋转
- **A/D** - 左右移动
- **S** - 软降
- **F** - 硬降
- **Q** - 暂存方块

**玩家 2（右侧）：**
- **↑** - 旋转
- **←/→** - 左右移动
- **↓** - 软降
- **Enter** - 硬降
- **M** - 暂存方块

## 安装和运行

### 从源码运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python tetris.py
```

### 使用可执行文件

从 [Releases](https://github.com/ganecheng-ai/tetris/releases) 页面下载对应平台的可执行文件：
- Windows: `Tetris.exe`
- macOS: `Tetris`
- Linux: `Tetris`

## 开发

### 构建可执行文件

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name Tetris tetris.py
```
