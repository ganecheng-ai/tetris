# 俄罗斯方块 - Tetris

经典的俄罗斯方块游戏，使用 Python 和 Pygame 开发。

## 游戏特性

- 精美的游戏界面和方块设计
- 10 个关卡，难度递增
- 简体中文界面
- 经典操作方式
- 影子方块提示
- 下一个方块预览
- 暂存方块功能（Hold）
- 连击（Combo）系统
- 多种游戏主题（经典、暗黑、明亮、霓虹）
- 方块皮肤系统（经典、霓虹、粉色、黄金）
- **双人对战模式** - 本地双人对战，互相攻击
- **多种游戏模式** - 经典、无尽、竞速、限时
- **成就系统** - 10 个成就等待解锁
- **数据统计** - 详细的游戏统计信息

## 游戏模式

### 经典模式
经典的俄罗斯方块玩法，10 个关卡挑战。每消除一定行数后升级，速度越来越快。

### 无尽模式
无限挑战模式，没有终点。每消除 10 行升一级，速度会逐渐加快，考验你的持久力！

### 竞速模式
以最快速度消除 40 行。时间就是你的对手，看看你能在多快的时间内完成挑战！

### 限时模式
在 2 分钟的时间内获得尽可能高的分数。时间结束后游戏结束，考验你的得分效率！

### 双人对战模式
本地双人对战，通过消除行数向对手发送垃圾行。先让对方无法放置方块者获胜！

**攻击规则：**
- 消除 2 行：发送 1 行垃圾
- 消除 3 行：发送 2 行垃圾
- 消除 4 行（Tetris）：发送 4 行垃圾
- 连击奖励：额外发送最多 5 行垃圾

## 操作说明

### 单人模式
- **← →** - 左右移动
- **↑** - 旋转方块
- **↓** - 软降（加速下落）
- **空格** - 硬降（直接落底）
- **C** - 暂存方块（Hold）
- **P** - 暂停/继续
- **R** - 重新开始
- **T** - 切换主题

### 双人对战模式

**玩家 1（左侧）：**
- **W** - 旋转
- **A/D** - 左右移动
- **S** - 软降
- **F** - 硬降
- **Q** - 暂存方块

**玩家 2（右侧）：**
- **↑** - 旋转
- **←/→** - 左右移动
- **↓** - 软降
- **Enter** - 硬降
- **M** - 暂存方块

## 安装和运行

### 从源码运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python tetris.py
```

### 使用可执行文件

从 [Releases](https://github.com/ganecheng-ai/tetris/releases) 页面下载对应平台的可执行文件：
- Windows: `Tetris.exe`
- macOS: `Tetris`
- Linux: `Tetris`

## 开发

### 构建可执行文件

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name Tetris tetris.py
```
