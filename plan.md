# 俄罗斯方块游戏开发计划

## 项目概述
使用 Python 语言开发一个俄罗斯方块游戏，要求画面精美，操作与经典俄罗斯方块游戏一致。
游戏界面支持简体中文，设置 10 个关卡。

## 已完成的功能 (v1.1.0)

### 核心游戏功能
- [x] 7 种标准方块 (I, O, T, S, Z, J, L)
- [x] 方块旋转和墙踢系统
- [x] 方块锁定和消行检测
- [x] 影子方块提示
- [x] 下一个方块预览

### 游戏系统
- [x] 10 个关卡系统（难度递增）
- [x] 计分系统（单/双/三/四行消除）
- [x] 升级系统（基于消除行数）
- [x] 游戏结束检测
- [x] 暂停/继续功能
- [x] 最高分记录系统

### 界面和体验
- [x] 简体中文界面
- [x] 精美的方块渐变效果
- [x] 操作说明显示
- [x] 关卡进度显示
- [x] 音效系统（硬降/消行/旋转/升级/游戏结束）

### 构建和发布
- [x] GitHub Actions 自动构建配置
- [x] 支持 Windows/macOS/Linux 平台
- [x] 自动发布到 GitHub Releases
