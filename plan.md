# 俄罗斯方块游戏开发计划

## 项目概述
使用 Python 语言开发一个俄罗斯方块游戏，要求画面精美，操作与经典俄罗斯方块游戏一致。
游戏界面支持简体中文，设置 10 个关卡。

## 已完成的功能 (v2.0.0)

### 核心游戏功能
- [x] 7 种标准方块 (I, O, T, S, Z, J, L)
- [x] 方块旋转和墙踢系统
- [x] 方块锁定和消行检测
- [x] 影子方块提示
- [x] 下一个方块预览

### 游戏系统
- [x] 10 个关卡系统（难度递增）
- [x] 计分系统（单/双/三/四行消除）
- [x] 升级系统（基于消除行数）
- [x] 游戏结束检测
- [x] 暂停/继续功能
- [x] 最高分记录系统

### 界面和体验
- [x] 简体中文界面
- [x] 精美的方块渐变效果
- [x] 操作说明显示
- [x] 关卡进度显示
- [x] 音效系统（硬降/消行/旋转/升级/游戏结束）

### 构建和发布
- [x] GitHub Actions 自动构建配置
- [x] 支持 Windows/macOS/Linux 平台
- [x] 自动发布到 GitHub Releases

### 双人对战模式 (v2.0.0 新增)
- [x] 本地双人对战
- [x] 攻击/垃圾行系统
- [x] 连击系统
- [x] 双人游戏界面
- [x] 玩家 1 控制 (WASD + QEF)
- [x] 玩家 2 控制 (箭头键 + M/0/Enter)

### 主菜单系统
- [x] 游戏模式选择（单人/双人）
- [x] 菜单导航界面

## 发布流程

1. 更新版本号
2. 创建 git 标签：`git tag v1.0.0`
3. 推送标签：`git push origin v1.0.0`
4. GitHub Actions 自动构建并发布
